📊 TradingView Indicator Code

คัดลอกโค้ดด้านล่างนี้ ไปวางในแถบ Pine Editor ของ TradingView แล้วกด 'Add to chart'

// This source code is subject to the terms of the Mozilla Public License 2.0
//@version=5
indicator("BossQuant EDCA Sniper (Multiplier)", overlay=true)

// --- Logic ---
sma200 = ta.sma(close, 200)
plot(sma200, color=color.blue, title="SMA 200")

is_crash = close < sma200 * 0.7
is_bear = close < sma200 and close >= sma200 * 0.7
is_normal = close >= sma200 and close <= sma200 * 1.3
is_bull = close > sma200 * 1.3

// --- Multiplier Rules ---
multiplier = 1.0
if is_crash
    multiplier := 5.0
else if is_bear
    multiplier := 2.0
else if is_bull
    multiplier := 0.5
else
    multiplier := 1.0

// --- Display Labels on Chart ---
if ta.crossover(close, sma200) or ta.crossunder(close, sma200) or ta.crossunder(close, sma200 * 0.7) or ta.crossover(close, sma200 * 1.3)
    txt = str.tostring(multiplier) + "X Buy"
    lbl_color = is_crash ? color.red : is_bear ? color.orange : is_bull ? color.green : color.blue
    label.new(bar_index, low - (low * 0.05), text=txt, style=label.style_label_up, color=lbl_color, textcolor=color.white, size=size.small)

// Background color for zones
bgcolor(is_crash ? color.new(color.red, 90) : na, title="Crash Zone")
bgcolor(is_bear ? color.new(color.orange, 90) : na, title="Bear Zone")
bgcolor(is_bull ? color.new(color.green, 90) : na, title="Bull Zone")

📖 คู่มือการเริ่มต้นใช้งาน

ขั้นตอนที่ 1: วางแผนเป้าหมายของคุณ

เปิดเครื่องมือ EDCA Calculator เพื่อเช็คว่าทุนตั้งต้นของคุณ ควรแบ่งเป้าหมายต่องวดเท่าไหร่ (เช่น มีทุน $10,000 หาร 25 งวด = เป้าหมายงวดละ $400) และลองใส่วันที่ย้อนหลังเพื่อดูผลลัพธ์

ขั้นตอนที่ 2: ติดตั้งสัญญาณลงกราฟ

คัดลอกโค้ด TradingView Script นำไปใส่ในกราฟ Bitcoin ของคุณ เพื่อให้ระบบขึ้นป้ายสัญญาณสีบอกโซนราคา (กระทิง/หมี/นรก) และบอกว่าต้องซื้อกี่ดอลลาร์ในจุดนั้น

ขั้นตอนที่ 3: ลงมือซื้อและบันทึกข้อมูล

เมื่อมีสัญญาณเตือน ให้ทำการเข้าซื้อบนกระดานเทรด (Binance) ตามจำนวนเงินที่ระบบบอก จากนั้นนำข้อมูลมากรอกลงใน Excel Journal ทุกครั้ง เพื่อให้ Excel ช่วยคำนวณต้นทุนเฉลี่ยและกำไรในพอร์ตให้คุณดูง่ายๆ
